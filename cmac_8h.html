Computes CMAC using 128-bit block cipher. Tag is significan_bits bit long and is stored in most significant bits of 128-bit output value.

Parameters
inset of several full 128-bit blocks to compute MAC for
blocksnumber of blocks in data
keyMAC key
significan_bitsnumber of bits in MAC, other bits will be zeroed. Possible values: see BCMLIB_CMAC_TAG_SIZE_* constants
outpointer to a 128-bit value, that receives the value of MAC
ciphercipher interface to use
